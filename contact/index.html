Contact

If you encounter bugs or other technical issues with the Protein Recognition Index (PRI) software package, please send an email to kuhnlab@msu.edu or use the GitHub Issue Tracker.

For questions about the Protein Recognition Index in context of the research article, please contact the publisher or corresponding author instead.
